# community-little-envoy

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Start mock server (json-server)
```
npm run mock
```
Mock API runs on `http://localhost:3001` by default.
